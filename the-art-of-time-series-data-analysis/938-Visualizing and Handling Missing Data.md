​
In the real world, when we are working with time series data, we can see a lot of missing information. The missing information can be because the sensor might be down at the time of capturing the temperature or any machine data or any other information. There can be a lot of other factors causing the
missing information.

This lesson will discuss how to deal with missing values in
time series data.
